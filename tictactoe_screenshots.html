Features Implemented
  • 2-player local game
  • X vs O marks
  • Row win detection
  • Column win detection
  • Diagonal win detection
  • Draw detection
  • Winning cells highlighted
  • Live scoreboard
  • Multi-round support
  • Overall winner at end
  • Invalid move handling
  • Play again option
